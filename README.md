# Arduino_ELwire
学科内でのファッションショーの時に使用するために、作成したELワイヤー８本を光らせるArduinoのプログラミングです。
３部構成になっていて、構成が終わる前にスキップ処理をして、次の構成にとべる機能も入れています。

ELワイヤーの光りかたは、下の４種類です。
同時点灯：８本のELワイヤーがすべて点灯
消灯：８本のELワイヤーがすべて消灯
ランダム：偶数と奇数番号のELワイヤーが交互に（）秒間隔で点灯と消灯を繰り返す
点滅：ELワイヤー８本すべてが（）秒間隔で点灯と消灯を繰り返す

第１部の構成（スイッチを1回押した時に発動するプログラミング）
➀３秒間：同時点灯　➁１６秒間：点滅　➂２５秒間：ランダム　➃６秒間：点滅　➄次にスイッチが押されるまで：消灯

第２部の構成
➀１４秒間：ランダム　➁２２秒間：同時点灯　➂１０秒間：点滅　➃３０秒間：同時点灯　➄７秒間：点滅
➅２６秒間：同時点灯　➆７秒間：点滅　➇２６秒間：同時点灯　➈１７秒間：点滅　➉次にスイッチが押されるまで：消灯

第３部の構成
➀ずっと：同時点灯
